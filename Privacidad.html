Políticas de privacidad - Terminos y Condiciones

Política de Privacidad

Información Recopilada

Al utilizar nuestra aplicación web de alerta contra sismos, recopilamos la siguiente información:

1. Información Personal:

• Nombre

• Dirección de correo electrónico

• Ubicación geográfica (coordinadas de latitud y longitud)

2. Datos de Dispositivo:

• Tipo de dispositivo

• Sistema operativo

• Identificador único del dispositivo

Uso de la Información

La información recopilada se utiliza para:

1. Proporcionar alertar personalizadas sobre sismos
2. Mejorar la precisión de las alertar utilizando datos de ubicación
3. Enviar comunicaciones importante sobre la aplicación y actualizaciones

Compartir Información

No compartimos información personal con terceros sin consetimiento expreso, excepto cuando sea necesario para cumplir con las leyes aplicables.

Seguridad

Tomamos medidas razonables para proteger la información personal contra accesos no autorizados o divulgación.

Términos y Condiciones

1. Uso Aceptable

El usuario se compromete a utilizar la aplicación web de alerta contra sismos de manera legal y ética. No se permite el uso indebido, el acceso no autorizado o la interferencia con el funcionamiento normal de la aplicación.

2. Responsabilidad

La aplicación proporciona alertas basadas en datos sísmicos, pero no garantiza la precisión ni la disponibilidad continua del servicio. El usuario es responsable de tomar las medidas necesarias para su seguridad en caso de alerta.

3. Actualizaciones

La aplicación se actualizará periódicamente para mejorar la funcionalidad y la seguridad. El usuario acepta instalar estas actualizaciones para continuar utilizando la aplicación.

4. Modificaciones en las Políticas

Nos reservamos el derecho de modificar estas políticas en cualquier momento. Las modificaciones entrarán en vigencia inmediatamente después de su publicación en la aplicación.

5. Terminación de Uso

Nos reservamos el derecho de suspender o terminar el acceso del usuario a la aplicación en caso de incumplimiento de estos términos y condiciones.

Consentimiento

Al utilizar nuestra aplicación, el usuario acepta las políticas de privacidad y los términos y condiciones establecidos anteriormente.

Si tienes requisitos específicos o consideraciones legales adicionales, te recomendaría buscar asesoramiento legal para personalizar estas políticas según las necesidades de tu aplicación y las leyes locales aplicables.
